The distribution and the tutorials are available at the PYXAID project on sourceforge.net

To obtain the package, click the "Browse Code" button or switch to the "Code" tab. In this new tab click the "Download snapshot" area. On Windows machines this will give you the option to download an archived (e.g. zipped) code tree, that includes Tutorials and level-3 code scripts.

Alternatively, on Linux-type systems, you can obtain a copy of git repository via:

git clone git://git.code.sf.net/p/pyxaid/code pyxaid-code
