Libann Tutorials


The "libann" is a module that implements operations with artificial neural networks (ANNs).

??.py

We start by creating a default ANN object: 

ANN = NeuralNetwork()
This will initialize some of the internal parameters of the ANN class methods (e.g., learning or printout), but other than that the object is not of a great use yet.

In the next step:

ANN.CreateANN([2,2,1])

we define the architecture of our ANN. This procedure will initialize the internals of the ANN representation (weights and biases). The dimensions of these variables will be defined and memory allocated. The initial values will be set at random.

In this particular example, we create an multilayer perceptron with 3 layers: 1 - input, 1 - hidden, and 1 - output. Each of these layers has a number of nodes (neurons): The first (input) layer has 2 nodes, so the input to the ANN should consist of 2 entries. The hidden layer also contains 2 neurons. The ouutput layer contains only 1 neuron, so the output (and the target/reference) values should consist of 1 entry. I use the term "entry" to indicate that an ANN can, in general, operate on various types of input - strings, numbers, or maybe vectors or matrices. However, for most cases, we will implying just numbers (integers and floats).

The content and all the internals of the ANN can be printed out with the "ShowANN()" function:

ANN.ShowANN()

Or exported into a file (specially formatted) using the "ExportANN" function, which also takes the name of the output file as an argument:

ANN.ExportANN("initial.ann")

The produced ".ann" file can be used to load an ANN with a sepcific state. For instance, the next lines use the "ImportANN" function to do just that - by loading the info about the previousely constructed ANN from a file.

ANN1 = NeuralNetwork()
ANN1.ImportANN("initial.ann")

The combination of the "ExportANN" and "ImportANN" functions can be very useful for saving the state of evolved (e.g., during the training ANN) and restarting the calculations from a particular ANN state.
